Customizing your own Scenario

The SWIM interface makes available a set of scientific models which you can customize using scenarios to project into the future. SWIM includes predefined scenarios to get you started; each predefined scenario defines the value for one or many input parameters of the model. Models have predefined scenarios grouped in categories, e.g., climate, technology and recovery.

Step 1: On the main navigation bar, place your pointer over the Models dropdown menu. A list of scientific models will be displayed. For this tutorial, select the Water Balance Model.

A landing page with infomation about the model will be displayed. To access the customization workflow, press the Create Scenario button located in the banner at the top of the page.

The customization workflow is a three-step process: Scenario Selection, Run Model and View Results.

Step 2: On the Select Scenarios step of the workflow, you can select one predefined scenario from each category. To select a scenario, press the representative image of the scenario. For this tutorial select a scenario combination from each category as follows: Climate - "Extreme Stress", Evaporation - "Default", and Reservoir Operation - "Minimum reservoir storage at 20% of capacity".

Step 3: Browse back to the climate scenarios. Each scenario contains a small description and a link for additional information. Press the Learn More... hyperlink. A popup box will be displayed. This box shows the defined values for the model parameters when selecting this scenario. Click on the Affected Model Inputs dropdown and select SanMarIn. The name, description and data plot of this input will be displayed. The units of the data values can be changed on the convert units dropdown menu.

Step 4 (optional): To further customize a scenario, the model's individual input parameters can be accesed through the Input Catalog (located on the top-right of the page). This catalog contains a listing of editable and non-editable input parameters. As an example, access the Inflows at San Marcial Gauges and press over the table row. A detailed view of the input will be displayed. The parameter values can be directly modified on the data table, and visualized as a line plot or bar plot.

Step 5: Go back to the select scenarios screen. Press the green Run Model button. A dialog box will be displayed with a Model Run Form. First, input a name for your custom scenario. Second, select your water-related role (i.e., what role identifies you when creating this scenario). Third, click on the Scenario Description text box. This text box will automatically include the information about the predefined scenario(s) used. Additional text can be included. Finally, you are required to agree with the website policies to run your model. You can learn more about these policies by clicking on the provided links. Once the form is complete, the Submit button will be enabled.

The results of executing your scenario will be available after a few seconds. The page automatically shows the View Results step once results are available. In contrast to the canned scenarios, the complete list of the model outputs is available to review.

Congratulations! Your are ready to SWIM on your own!
