Client Requirements

The following is a list of recommended requirements for the SWIM interface to run smoothly on your device:

  • SWIM is compatible with most up-to-date web browsers.
  • SWIM is tested using the latest Firefox Web Browser for Desktop (version >= 105.0.2).
  • SWIM is currently incompatible with Firefox for Android.
  • Medium to high resolution screen (e.g tablet size and above).
